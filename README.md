# Simple Scripts to Run whisper.cpp locally

## The current setup_script.sh only support Mac.

## setup_script.sh Requirements:

- Apple Silicon 
- 5GB space (You can modify the scripts to use smaller model)
- Internet connection

##
```shellscript
sh setup_whisper.sh
```

## 

```shellscript
sh setup_whisper.sh
```

For windows

```powershellscript
setup_whisper.ps1
```
