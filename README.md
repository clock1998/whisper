# Simple Scripts to Run whisper.cpp locally

## The current setup script only support Mac.

## Requirements:

- Apple Silicon 
- 5GB space (You can modify the scripts to use smaller model)
- Internet connection

## Run setup_whisper.sh first to setup everything automatically.
```shellscript
sh setup_whisper.sh
```
## Run
```shellscript
sh setup_whisper.sh
```

```powershellscript
setup_whisper.ps1
```
